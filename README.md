# 🍔 One Bite Cafe Website

![One Bite Logo](https://franchisetrade.in/wp-content/uploads/2018/12/one-bite-logo.png)

A modern, fully responsive restaurant website for **One Bite Cafe**, built using **HTML, CSS, and JavaScript**. The website features an animated user interface, category-based menu navigation, interactive food sections, and direct contact options for customers.

## ✨ Features

- 🎬 Animated splash screen with One Bite logo
- 📱 Fully responsive design for mobile, tablet, and desktop
- 🍔 Category-based menu system
- 📂 Expandable menu categories with item counters
- 🥗 Veg / Non-Veg filtering
- 🔍 Easy menu navigation
- 🎨 Modern UI with smooth animations
- 📸 Food gallery section
- ⭐ Customer reviews section
- 🎁 Special offers section
- 📞 One-click Call button
- 💬 WhatsApp integration
- 📷 Instagram integration
- ⬆️ Back-to-top button
- ⚡ Fast and lightweight

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- AOS Animation Library
- GSAP Animations
- Font Awesome Icons
